

## Prompt Reescrito

---

**Adicione uma seção de "Lojas que confiam no CarPost" com carrossel de logos de clientes**

Crie uma nova seção no site exibindo logos de lojas parceiras/clientes em um carrossel horizontal contínuo (infinite scroll), posicionada logo após a seção Hero.

**Requisitos:**

1. **Layout:** Carrossel horizontal com scroll automático infinito (marquee style), sem botões de navegação
2. **Conteúdo:** Exibir 8-12 logos de lojas fictícias em escala de cinza, com efeito hover que revela a cor original
3. **Design:**
   - Fundo neutro e limpo para destacar os logos
   - Título curto acima: "Lojas que já vendem mais com o CarPost" ou similar
   - Logos com opacidade reduzida (60-70%) que aumenta no hover
   - Espaçamento uniforme entre logos
4. **Animação:** Movimento contínuo suave da direita para esquerda, pausando ao passar o mouse
5. **Responsivo:** Adaptar quantidade visível de logos para mobile (3-4) e desktop (6-8)
6. **Performance:** Usar CSS animation para o scroll infinito, sem dependências pesadas
7. **Estilo:** Seguir o design system atual do site (bordas arredondadas, sombras suaves, estilo SaaS moderno)
